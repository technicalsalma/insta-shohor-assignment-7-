3: What are the difference between Local Storage and Session Storage?

Answer:

LocalStorage
SessionStorage
1. Local Storage use for long term use.
1. Session Storage use for temporary use
2. Local storage data will not expire. It will remain even after browser restart and survive OS reboot too.
2. Session Storage is lost the moment the browser is closed
3. Data stored in the local storage is also available after the window has been closed.
3. When the browser window is closed, the session storage associated with that window is deleted.
